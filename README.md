# Excavator
Scripts for the small RC-excavator. If you want to have miniature excavator with remote driving, end effector control plus all kinds of nice little addons you've come to the right place!

-Pi5
-Pi Pico (or similar)
-3x Adafruit ISM330DHCX IMU
