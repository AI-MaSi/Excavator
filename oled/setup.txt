OLED Display Module
===================

Part of MASI robot system. Full setup handled by:  sudo ../setup.sh

Manual control:
  oled-start   - Start display service
  oled-stop    - Stop display service
  oled-status  - Check service status

Wiring (corner of Pi, pins 38-40):
  Pin 38 (GPIO 20) -> SDA
  Pin 40 (GPIO 21) -> SCL
  Pin 39 (GND)     -> GND
  Pin 17 (3.3V)    -> VCC

I2C bus: 4 (virtual, ~100kHz)
I2C address: 0x3D

Verify: i2cdetect -y 4
Logs:   journalctl -u oled.service -f
